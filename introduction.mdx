## What is OrgStack?

OrgStack is a production-ready multi-tenant organization management platform that provides the foundation for building secure, scalable SaaS applications. Built with Spring Boot and Angular, it implements enterprise patterns for tenant isolation, authentication, and authorization out of the box.

  <Card title="Enterprise security" icon="shield-halved">

## Key features

### Strict tenant isolation

OrgStack ensures complete data separation between organizations. Every entity is scoped to a tenant, preventing cross-tenant data leaks.

```java BaseEntity.java
@MappedSuperclass
@EntityListeners(AuditingEntityListener.class)
@Getter
public class BaseEntity {
    @Column(nullable = false, updatable = false)
    private UUID id;
    
    @CreatedDate
    @Column(nullable = false, updatable = false)
    private Instant createdAt;
    
    @LastModifiedDate
    @Column(nullable = false)
    private Instant updatedAt;

    protected BaseEntity() {
        this.id = UUID.randomUUID();
    }
}
```

### Role-based access control

Fine-grained permissions system that allows you to control what users can do within their organization.

### JWT authentication

Secure, stateless authentication using JSON Web Tokens, integrated with Spring Security.

### Audit logging

Automatic tracking of creation and modification timestamps on all entities using JPA auditing.

```java JpaConfig.java
@Configuration
@EnableJpaAuditing
public class JpaConfig {
}
```

- **Spring Security** - Enterprise-grade security
- **Spring Data JPA** - Database abstraction with Hibernate
